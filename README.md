# Deep Maze
 
A randomized maze game made using pure HTML, Javascript, CSS and Canvas.

## Demo

https://lhfdv.github.io/DeepMaze/

## Rules

Controls:
* ^  -  Move up
* v  -  Move down
* <--  -  Move left
* -->  -  Move right
* R  -  Restart the game
* DEL - Dark Mode

How to play:

* Move the character (green dot) towards the red dot before the timer runs out.
